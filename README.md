## 开启 GitHub Pages

1. 进入仓库 **Settings**
2. 左侧找到 **Pages**
3. Source 选择 `Deploy from a branch`
4. Branch 选择 `main`，目录选择 `/ (root)`
5. 点击 Save，等待 1-2 分钟生效

## 本地预览

```bash
cd image-processing-notes
python3 -m http.server 8080
# 浏览器访问 http://localhost:8080
```
