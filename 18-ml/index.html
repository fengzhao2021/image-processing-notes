传统机器学习

深度学习的地基。这一层讲的所有概念在深度学习里都会直接用到，只是规模从几个参数变成了几千万个参数。

CORE CONCEPTS

核心概念总览

回归
在数据点上找一条曲线，用已知数据预测未知数值。比如用ISO和曝光时间预测噪点评分。
损失函数
衡量当前曲线有多差。预测值和真实值之间的差距，差距越小模型越好。通常用均方误差：Σ(预测-真实)²
梯度下降
往损失减小的方向调整参数，就像蒙眼在山上往最陡的方向下山。重复直到损失足够小。
学习率
每次梯度下降走多大步。太大会越过最低点来回震荡，太小收敛太慢。
分类
预测离散类别而不是连续数值。最简单的是逻辑回归：线性组合→Sigmoid压缩到0~1→输出概率。
过拟合
训练误差很小但测试误差很大，把噪点当成规律学进去了。就像死记硬背考试，原题全对换个说法就不会。
训练集/测试集
数据分成两部分，训练集用来训练，测试集完全不碰只用来评估。测试集误差才是模型真实能力的体现。
Mini-batch
每次用一小批数据（32张）计算梯度，更新参数。比全量快，比单张稳定。Batch Size越大越稳定但越占显存。
Epoch
把全部训练数据过一遍叫一个Epoch。训练神经网络通常需要几十到几百个Epoch。
特征工程
把原始数据加工成更适合模型学习的形式：归一化、构造新特征、对数变换、One-Hot编码等。深度学习不太需要，CNN自动学习特征。
GRADIENT DESCENT

梯度下降

参数更新公式
新参数 = 旧参数 - 学习率 × 损失函数对参数的导数

导数告诉你：当前方向损失是增大还是减小
减去导数   ：往损失减小的方向走一步
重复       ：直到损失不再减小

批量梯度下降

每次用全部数据计算梯度。方向准确但极慢，数据量大时不可用。

随机梯度下降 SGD

每次用1张图。极快但方向噪点大，像喝醉的人下山，总体往下走但路径很乱。

Mini-batch（实际使用）

每次用32张图。速度快，方向比单张稳定，是实际训练神经网络的标准做法。

OVERFITTING

过拟合与解决方法

核心问题

模型太复杂，把训练数据里的噪点和特殊情况都记住了，而不是学到真实规律。训练误差很小但面对新数据表现很差。

更多数据

最直接有效。数据越多，噪点影响越小，模型越难把噪点当规律学。

降低模型复杂度

用更简单的模型，减少参数数量，减少过拟合空间。

正则化

在损失函数里加惩罚项：总损失 = 预测误差 + λ×参数大小。惩罚参数太大，强迫模型保持简单。

Dropout

深度学习专用，训练时随机关掉神经元，强迫模型不依赖任何单个神经元。Layer 05细讲。

RELATION TO DEEP LEARNING

和深度学习的关系

本质完全相同，规模不同
传统机器学习：
  几个到几百个参数
  手工设计特征
  损失函数 → 梯度下降 → 更新参数

深度学习：
  几百万到几百亿个参数
  自动学习特征
  损失函数 → 梯度下降 → 更新参数（5000万个）

原理完全一样，规模不同
